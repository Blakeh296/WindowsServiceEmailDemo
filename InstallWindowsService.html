https://docs.microsoft.com/en-us/dotnet/framework/windows-services/walkthrough-creating-a-windows-service-application-in-the-component-designer

Sending Email from C#

The procedure for sending e-mail from C# is actually pretty easy:
1. Create the message object.
2. Create the SMTP client.
3. Create the network credentials (username, password)
4. Send the credentials to the SMTP client.
5. Set the message properties and content.
6. Tell the SMTP client to send the message object.
7. Record the result somewhere.


using System.Net;
using System.Net.Mail;

    MailMessage msg = new MailMessage();
    SmtpClient mailClient = new SmtpClient("mail.yourserver.com", 587);
    NetworkCredential emailCred = new NetworkCredential("<user name>", "<password>");

    try
    {
        mailClient.EnableSsl = true;
        mailClient.Credentials = emailCred;
        msg.From = new MailAddress("you@fromaddress.com");
        msg.To.Add("them@toaddress.com");
        msg.Subject = "Application Mail";
        msg.Body = "The system was started at " + DateTime.Now.ToString();
        mailClient.Send(msg);
        Console.WriteLine("Email was sent successfully");
    }
    catch (Exception ex)
    {
        // output error somewhere
    }

The code above shows an outgoing connection.

Using this knowledge of Windows services and sending e-mails, you could create a Windows service that notifies you of events on the system by e-mail.

Hiding the user password entry

One of my examples tonight was allowing the user to enter a password through a console window. In this case, you would probably want to hide the password input.

bool passwordEntry = true;

while (passwordEntry)
{
    ConsoleKeyInfo key = Console.ReadKey(true);
    if (key.Key == ConsoleKey.Enter)
        passwordEntry = false;
    else
        passWord += key.KeyChar;
}

The above code intercepts the user's key input one key at a time, stops it from going to the Console window and adds it to a password string. When the user presses Enter, that signals the end of the string.
